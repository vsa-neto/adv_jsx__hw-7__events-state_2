

# <span style="color:orange">Events state</span>

приложение для интернет-магазина, которое позволяет отображать товары в двух разных форматах: в виде сетки из карточек или в виде вертикального списка
===

## Состав компонентов:

__1. Store (компонент с состоянием):__
* это главный компонент, который управляет логикой приложения. Он хранит в себе состояние (выбранный вид отображения) и решает, какой из компонентов — CardsView или ListView — должен быть отрендерен.

__2. IconSwitch (компонент без состояния):__
* отвечает за отображение иконки-переключателя. Он не имеет своего состояния и получает все необходимые данные (название иконки и обработчик клика) от родителя.
__3. CardsView (компонент без состояния):__
* отвечает за отображение товаров в виде сетки. Получает массив товаров и отрисовывает их с помощью компонента ShopCard

__4. ListView (компонент без состояния):__
* отвечает за отображение товаров в виде списка. Получает массив товаров и отрисовывает их с помощью компонента ShopItem.

__5. opCard и ShopItem (компоненты без состояния):__ 
* отвечают за отрисовку одного товара в формате карточки и строки списка соответственно.


## Функциональные требования

__1. Инициализация:__
* Компонент Store при инициализации содержит полный список товаров и устанавливает вид по умолчанию (например, 'view_module' — карточки)

__2. Взаимодействие Store и IconSwitch:__
* Store передаёт в IconSwitch иконку для отображения через пропс icon (например, 'view_list', если текущий вид — карточки).
* Store передаёт в IconSwitch функцию для переключения вида через пропс onSwitch

__3. Обработка действий пользователя:__
* При нажатии на иконку в IconSwitch вызывается переданный обработчик onSwitch.

__4. ВУправление состоянием:__
* Обработчик onSwitch в компоненте Store должен изменять внутреннее состояние (state), переключая его между 'view_module' и 'view_list'.

__5. Условный рендеринг:__
* SВ зависимости от текущего состояния, Store должен отображать либо компонент CardsView, либо ListView.
* Store передаёт массив товаров в соответствующий пропс (cards для CardsView или items для ListView).
